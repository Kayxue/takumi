## ImageResponse

The `ImageResponse` class is a high-level API for generating images from JSX. It extends the standard [Response](https://developer.mozilla.org/en-US/docs/Web/API/Response) object.

### Options

The options object for `ImageResponse` combines standard [ResponseInit](https://developer.mozilla.org/en-US/docs/Web/API/Response/Response#options) properties with Takumi-specific rendering options.
